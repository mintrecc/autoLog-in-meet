# AutoLog-In-Meet

Скрипт для автоматического подключения к онлайн-лекциям Google Meet на основе расписания учебного заведения.

## Функционал
- Парсинг расписания с сайта e-rozklad.duikt.edu.ua.
- Автоматическое открытие Google Chrome с выбором конкретного профиля.
- Использование PyAutoGUI для отключения микрофона/камеры и нажатия кнопки входа.
- Блокировка перехода компьютера в спящий режим во время работы скрипта.

## Структура проекта
- `main.py` — точка входа, содержит основной цикл проверки времени и управление состоянием питания ПК.
- `parser.py` — логика авторизации на сайте и получения актуального списка пар.
- `binds.py` — база ссылок на конференции и логика взаимодействия с браузером.

## Технологии
- Python 3.x
- Requests / BeautifulSoup4 (парсинг)
- PyAutoGUI (автоматизация действий)
- Webbrowser (управление браузером)
- Ctypes (системные вызовы Windows)
